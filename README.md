# RecruitAi — Campus Recruitment Management System

A full-stack web application that automates campus placement for students, recruiters, and college admins.

## 🔗 Live Demo
**[https://recruitai-pu35.onrender.com](https://recruitai-pu35.onrender.com)**

## 👥 User Roles

- **Student** — Register, browse jobs, get auto-shortlisted, track application status
- **Recruiter** — Post jobs, auto-shortlist eligible students, manage hiring pipeline
- **College Admin** — Approve recruiters, manage students, view placement analytics

## ✨ Key Features

- Auto-shortlisting based on CGPA, branch and skills when a job is posted
- JWT-based authentication with role-based access control
- Real-time application tracking: Applied → Shortlisted → Interview → Selected
- Recruiter approval workflow managed by college admin
- Coding test creation and assignment
- Placement analytics dashboard
